# Supabase Custom SMTP Setup Guide (Gmail)

Supabase의 기본 이메일 서비스는 하루 전송량이 매우 제한적입니다.
원활한 테스트와 서비스를 위해 **Gmail SMTP**를 연동하는 것을 권장합니다.

## 1. 구글 앱 비밀번호 발급
Gmail 계정의 진짜 비밀번호가 아닌, **전용 앱 비밀번호**가 필요합니다.

1. [Google 계정 관리](https://myaccount.google.com/) 접속
2. **보안 (Security)** 탭 클릭
3. **2단계 인증 (2-Step Verification)**이 켜져 있어야 합니다. (안 켜져 있다면 켜주세요)
4. **팁**: 메뉴를 찾기 힘들다면, 상단 **검색창**에 `앱 비밀번호` 또는 `App passwords`라고 검색하세요.
5. 검색 결과에서 **앱 비밀번호**를 선택합니다.
6. **앱 이름**: `Supabase` (아무거나 입력) -> **만들기**
6. 생성된 **16자리 비밀번호** (예: `abcd efgh ijkl mnop`)를 복사해두세요.

## 2. Supabase 설정 입력
방금 띄우신 Supabase 설정 창에 아래 내용을 입력하세요.

| 항목 | 입력값 | 설명 |
| :--- | :--- | :--- |
| **발신자 이메일** | `본인지메일@gmail.com` | 실제 보내는 사람 주소 |
| **발신자 이름** | `The Manager` | 받는 사람에게 표시될 이름 |
| **호스트 (Host)** | `smtp.gmail.com` | 구글 메일 서버 주소 |
| **포트 (Port)** | `465` | SSL 포트 번호 |
| **사용자 이름** | `본인지메일@gmail.com` | 구글 아이디 |
| **비밀번호** | `(아까 복사한 16자리 코드)` | **앱 비밀번호** (로그인 비번 X) |

## 3. 저장 및 확인
- **최소 간격**: `1`초 (테스트용) 또는 `60`초 (스팸 방지용)
- **저장** 버튼 클릭

이제 이메일 전송 제한이 대폭 늘어나며, 로봇 테스트도 막힘없이 수행할 수 있습니다.

## 4. (필수) 이메일 템플릿 수정
Supabase 기본 템플릿에는 "숫자 코드"가 없고 "링크"만 있습니다. 
우리는 6자리 숫자 코드를 입력받으므로 템플릿을 수정해야 합니다.

1.  **Authentication** -> **Email Templates** 메뉴로 이동
2.  **Confirm Your Email** (또는 Sign Up) 템플릿 클릭
3.  **Message Body** 내용을 다 지우고 아래 내용으로 교체(또는 `{{ .Token }}`을 추가)하세요:

```html
<h2>인증 코드: {{ .Token }}</h2>
<p>이 코드를 앱에 입력하세요.</p>
```

4.  **Save** 버튼 클릭

이제 이메일에 6자리 숫자가 포함되어 올 것입니다.
