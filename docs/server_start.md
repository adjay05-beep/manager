# 서버 시작 가이드

## 🚀 좀비 프로세스 없이 서버 시작하기

이제부터는 **절대 `python main.py`를 직접 실행하지 마세요!**

대신 아래 스크립트를 사용하세요:

### Windows PowerShell
```powershell
.\start.ps1
```

### Windows 명령 프롬프트 (CMD) / 더블클릭
```
start.bat
```

## ✨ 자동으로 해주는 일

1. **포트 8555을 사용하는 이전 프로세스 자동 종료**
2. **좀비 Python 프로세스 정리**
3. **깨끗한 상태로 새 서버 시작**

## 📝 사용 방법

1. `Project A` 폴더에서 터미널 열기
2. `start.bat` 더블클릭 **또는** 터미널에서 `.\start.ps1` 실행
3. 끝! 

## ⚠️ 주의사항

- **절대 여러 개의 터미널에서 동시에 실행하지 마세요**
- 서버를 종료하려면 `Ctrl+C` 누르세요
- 개발 중에는 항상 `start.bat`만 사용하세요

## 🐛 여전히 좀비 프로세스가 생긴다면

수동으로 모든 Python 프로세스 종료:
```powershell
taskkill /F /IM python.exe
```

그 다음 다시 `start.bat` 실행
